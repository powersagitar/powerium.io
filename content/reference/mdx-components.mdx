These components are available globally in every `.mdx` file — no import needed.
Each component has its own page with a full reference and live examples.

| Component             | Description                                               |
| --------------------- | --------------------------------------------------------- |
| `<ArticleList />`     | Sorted article list from a content directory              |
| `<ArticleListItem />` | Single article row (used internally by `<ArticleList />`) |
| `<TableOfContents />` | Scroll-aware heading navigation                           |
| `<Spacer />`          | Vertical whitespace                                       |

<ArticleList dir="components" recursive={false} />

## Adding new components

1. Create the component in `src/components/mdx/`.
2. Export it from `src/components/mdx/index.tsx` and add it to the
   `mdxComponents` map.
3. Add a page in `content/components/` with a synopsis, reference table, and
   examples.
4. Update `content/reference/mdx-components.mdx` to include the new row in the
   table above.

---
title: table-of-contents
description:
  Auto-detects headings and renders a linked, scroll-aware navigation list.
---

`::table-of-contents` reads all `h2` and `h3` headings from the current article
and renders a linked navigation list. The active heading is highlighted as you
scroll using `IntersectionObserver`. Returns nothing if the page has no
qualifying headings.

Maps to the `<TableOfContents />` React component via the kebab-case →
PascalCase directive convention. Runs client-side.

## Reference

```mdx
::table-of-contents
```

No attributes. Place it anywhere in the MDX body — typically near the top of
long articles, before the first heading.

---
title: TableOfContents
description:
  Auto-detects headings and renders a linked, scroll-aware navigation list.
---

`<TableOfContents>` is a client component that reads all `h2` and `h3` headings
from the current article and renders a linked navigation list. The active
heading is highlighted as you scroll using `IntersectionObserver`. Returns
`null` if the page has no qualifying headings.

## Reference

```mdx
<TableOfContents />
```

No props. Place it anywhere in the MDX body — typically near the top of long
articles, before the first heading.
